# Desafio Tech - Biopark

## Instalação
Para instalar as dependências do projeto:

`npm install`

Para inicializar o projeto:

`npm start` 

## Configuração
Para inicializar o servidor:

`node server.js`

Para inicializar o banco de dados:

        host: 'localhost',
        user: 'root',
        password : '123456',
        port : 8080,
        database:'testdb'
